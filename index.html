The .remove() method

The .remove() method is used to remove a selected element or node from the DOM.
The .html() method

The .html() method is used to insert text in a DOM node or element, but additionally unlike the .text method, can be used to insert fully formed HTML elements

The .attr() and .property() methods

The .attr() and .property() methods are used to add an attribute or property to a selected element. It accepts two arguments as input; the attribute to be added and the value to be assigned to it.

The .style() method

The style() method is used to manipulate the style properties of a selected element or DOM node.

It accepts two arguments as input, which are the property to be set, and the value to be assigned to it.
